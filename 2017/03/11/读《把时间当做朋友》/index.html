读《把时间当作朋友·李笑来》

发表于 2017-03-11 更新于 2020-03-15 分类于 读书笔记 Valine：

人生的幸运在于能够“用正确的方式做正确的事情”。

这是幸运的，很多时候我们没那么幸运，我们不知道什么是正确的事情，不知道什么又是正确的方式，所以这就是你不行动的的理由吗？

长期来看，我们都是要死的。但是明显有些人，在用另外一种状态生活，他们从容优雅，善于化解各种压力，安静地去做他们认为应该做的事情，并总是有所成就。

都说人要“慎独”，其实人要“慎群”，我们常常是被集体带着做一些很傻的事情，心没有静下来。

人们觉得自己正在做自己不喜欢的事情，然而，真的是他们以为的那样吗？不客气地说，99%的情况下并非如此。首先，这些人并不是对正在做的事情没有兴趣，而是没能力把目前正在做的事情做好。最终没有人会喜欢自己做不好的事情。

可能比率没有99%那么高，但是我觉得很有道理，很多时候其实都是因为没有把一件事做好，而后开始讨厌这件事。如果一件事你能做好，做到比谁都好，你根本不会对这件事没兴趣，所以往往不是有兴趣才能做好，而是做好了才会有兴趣。

方法固然重要，但是比起“用功”来说，方法可以忽略不计。

听的明白的话，这句话说的实在是对，类似的一句话是网上很流行的“大部分人的努力程度，还轮不到拼天赋”。不用钻牛角尖的，大多数事情的失败都是因为根本不够用功，没能坚持，不过我想，一般人应该用不了这个功吧，所以一般人是一般人。

大多数人连一次“坚持到底最终成功”的经验都没有

来，我们也体验一把。

除了“试错”、“观察”、“阅读”之外，“思考”，准确地说，“正确地思考”，才是获取真正意义上的知识的主要手段。

错误要总结，观察要记录，读书要分享。

马可波罗在700多年前，在没有“金山词霸”，不懂“艾宾浩斯曲线”，既不“逆向”，也不“疯狂”的情况下，学会了地球上最无从捉摸的、容易忘记、难以学习的语言————中文

一天到晚都在说学英语，学英语，我们这哪里是在学英语啊，搞笑了。

与其浪费时间抱怨不公平，还不如花点时间做些真正能改变点什么的努力。

想起高中时考试，有些人会作弊，然后有的人不愿意作弊，然后就抱怨不公平，其实确实是不公平，但是对于不作弊的你来说，自己考的足够好才是最重要的事。

一个人心智能力一旦正常开启，就会发现自己在这个信息唾手可得的世界里，只要正常地努力，并且有耐心和时间做朋友，很容易成为至少一个领域的专家。努力并不像传说中那么艰苦，只不过是“每天至少专心学习工作6小时”；耐心却远比大多数人想象的要大，“要与时间相伴断则至少5年，长则20年”。

这个时代，只要我们想学习，到处都为你敞开大门，网络上的很多课程（比如Coursera）和资源都是免费或者是低价的，图书馆条件又那么好，其实，最难的是如何在这个试图把一切碎片化的到处充满诱惑的浮躁时代，静下心来，这就得运用心智了。

生活的智慧就在于，集中精力改变那些能够改变的。专心打造自己，把自己打造成一个优秀的人，一个有用的人，一个独立的人，比什么都重要。

这得先从生活能够自理做起。

事实上，真正的关心最终只有一个表现：为之心甘情愿地花费时间，哪怕浪费时间。

额，李老师，所以你是要告诉我“陪伴是最长情的告白”吗？0.0

一个人的幸福程度，往往取决于他多大程度上可以脱离对外部世界的依附。

嗯，一样，这得先从脱离生活不能自理做起。

想提高你的社交质量，你最好多花时间和精力把自己变得更加优秀

周国平说一切交往的质量都取决于交往者本身的质量。唯有两个灵魂充实丰富的人，才可能有真正动人的爱情和友谊。

早一些开始准备，早一点，再早一点，越早一点，你与时间这个朋友之间就会变得越亲密；与你更加亲密的这个朋友会让你的每一天都多那么一点点从容；而这一点点从容，会让你爱上时间这个伙伴。

我不停地告诫自己，不要因为害怕不够完美而拒绝开始，正确的做法是马上行动，先做起来，先做出个大概来，然后逐步迭代，优化升级，这样才能把事情漂亮地完成。

人为什么随着年龄变大，觉得时间过得越来越快了呢？

对一个5岁的孩子来说，未来的一年相当于他已经度过的人生的20%；而对于一个50岁的人说，未来的一年只相当于他已度过的人生的2%。

时间的朋友，就是拿一生去做那些时间越长价值就越大的事。

这句摘自罗辑思维-时间的朋友跨年演讲的广告宣传语。
